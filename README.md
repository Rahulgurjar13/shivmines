# Shiv Minerals - Premium Silica Mining & Supply

A modern website for Shiv Minerals, a leading silica mining company.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Build

```sh
# Production build
npm run build

# Preview production build
npm run preview
```

## Deployment

Deploy to any static hosting service like Vercel, Netlify, or your own server.
